Kieran's Symbolic Calculator

Edit the equation in the first text box below and enter the variable you'd like to solve for in the small text box below that one. Make sure you only have one symbol in the second text box, but it can be a symbol with a subscript or a greek letter. Press "\" followed by some TeX command to enter greek letters or unlock hidden functionality (Integrals?) This is still very much a development build, please let me know if you have any suggestions
